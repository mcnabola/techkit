Arduino IDE

The arduino ide is a way for our computer to connect with the ESP board. We can code in C or arduino C(?) and then load the code onto the board using the USB device that came in the TechKit package.

Installing, https://www.arduino.cc/en/software They have downloads for mac windows and Linux on this page. If you are on linux you can probably download it from your package manager too in one line. pacman -S arduino , for Arch or Manjaro Systems. sudo pacman -S arduino-avr-core (download this too)

Hello, World!

After you get the arduino ide installed and opened, you can now start preparing the IDE to work with the ESP. By default it doesnt work with ESP's as it was originally built for Arduinos but the software eventually transformed to work with ESP's too.

What do you gotta do? Change the board manager preferences to allow other boards to work. Select one of those other boards. Change the default bord you are working with and finally for me, I selected the serial port I wanted to send data over.

  1. File>Preferences>Copy "http://arduino.esp8266.com/stable/package_esp8266com_index.json" into the 'Additional board managers URLs.>click OK
  2. Tools>Board>click the arrow>Boards Manager>Search>"ESP8">ESP8266 should show up>click INSTALL
  3. Tools>Board>Generic ESP8266 board
  4. Tools>Port> any of the options that have the word "usb" in them.

For the last step of the process you need to have drivers for the serial port connention or for linux users add your user to 2 user groups.Good resource for Linux
